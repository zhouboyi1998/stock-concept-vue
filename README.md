### ⌛ 开始

* 选择一个 Node 包管理器

#### 安装依赖

```bash
npm install
```

```bash
yarn install
```

```bash
pnpm install
```

#### 运行服务

```bash
npm run dev
```

```bash
yarn dev
```

```bash
pnpm dev
```

#### 构建预览包

```bash
npm run preview
```

```bash
yarn preview
```

```bash
pnpm preview
```

#### 构建部署包

```bash
npm run build
```

```bash
yarn build
```

```bash
pnpm build
```
