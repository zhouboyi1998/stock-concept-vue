### ⌛ Start

* Choose a Node package manager

#### Install dependencies

```bash
npm install
```

```bash
yarn install
```

```bash
pnpm install
```

#### Run serve

```bash
npm run dev
```

```bash
yarn dev
```

```bash
pnpm dev
```

#### Build preview package

```bash
npm run preview
```

```bash
yarn preview
```

```bash
pnpm preview
```

#### Build deployment package

```bash
npm run build
```

```bash
yarn build
```

```bash
pnpm build
```
